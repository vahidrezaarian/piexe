# Piexe – Quick & Private Image Text/QR/Barcode Extractor
Piexe is a lightweight, completely offline Windows app that helps you instantly extract text, QR codes, and barcodes from any image.

## Key features:
### Analyze screenshots you just took
### Open images directly with Piexe (right-click → Open with)
### Extracts all readable text and shows it as selectable items
### Automatically detects QR codes & barcodes → shows decoded content

### One-click actions for each extracted item:
- Copy to clipboard
- Open URL in browser (if it's a link)
- Open folder in Explorer (if it's a valid path)
- Open terminal in that folder (normal or as Administrator)

### No internet connection • No data collection • No telemetry • 100% local processing

### Perfect for quickly grabbing text from memes, receipts, documents, whiteboards, or scanning any kind of code without uploading anything anywhere.
